Task Description

Hello! Looks like you're up and running. Here's what to do next.

Modify this Django application such that a BED file can be uploaded here. The uploader should, at minimum:

  • Save the genomic regions specified in the BED file to database via Django's ORM.
  • Render a graphical summary of the regions uploaded. plotly.js would be a reasonable tool for this, but use what you like. It's okay if your visualization doesn't seem very "useful," since this may be your first exposure to this kind of data.
  • Be well tested!

Example BED files can be found here and here, but know that we'll be testing your code with more than just these files! Feel free to modify the existing code however you like, so long as the above requirements are met. This is only a starting point.
